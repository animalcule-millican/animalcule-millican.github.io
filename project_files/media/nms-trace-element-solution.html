NMS trace element solution

  • Add 100 ml DI-H~2~O to beaker
  • Then add the following:
    • 50 mg FeSO~4~ * 7H~2~O
    • 40 mg ZnSO~4~ * 7H~2~O
    • 2 mg MnCl~2~ * 7H~2~O
    • 5 mg CoCl~2~ * 6H~2~O
    • 1 mg NiCl~2~ * 6H~2~O
    • 1.5 H~3~BO~3~
    • 25 mg NaEDTA
  • Filter sterilize
  • Store sealed at 4^o^C
