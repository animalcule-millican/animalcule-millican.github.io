Azobacter media components

Carbon solutions:

- Other carbon sources can be substituted as needed. 
- Verify solubility of alternative carbon sources and adjust carbon solution and media recipe accordingly

Glucose:

  1. Add 100ml DI-H~2~O to flask
  2. Add the following carbon sources to flask:
    • 50g glucose to flask
  3. filter sterilize solution
  4. Store at 4^o^C

Mannitol:

  1. Add 100ml DI-H~2~O to flask
  2. Add the following carbon sources to flask:
    • 50g mannitol to flask
  3. filter sterilize solution
  4. Store at 4^o^C
