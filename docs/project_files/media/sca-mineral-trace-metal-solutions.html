Components of SCA media

Mineral and trace metal solutions for SCA media

solution A

  1. 20 g KNO3 (potassium nitrate)
  2. 20 g NaCl (sodium chloride)
  3. 20 g K2HPO4 (potassium phosphate dibasic)
  4. 1000 mL DI-H2O
  5. Filter sterilize solution

solution B

  1. 0.5 g MgSO4 (magnesium sulfate)
  2. 0.1 g FeSO4 (ferric sulfate)
  3. 0.01 g ZnCl2 (zinc chloride)
    • OR 0.02g ZnSO4 (zinc sulfate heptahydrate)
  4. 1000 mL DI-H2O
  5. Filter sterilize solution
