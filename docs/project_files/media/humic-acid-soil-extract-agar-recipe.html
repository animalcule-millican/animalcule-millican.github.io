Humic acid soil extract agar

Variation of humic acid agar supplemented with soil extract solution

Recipe:

  1. Prepare soil extract solution
  2. Add 968 ml of nanopure H~2~O
  3. Add 30 ml soil extract solution
  4. Add the following to water-soil extract solution:
    • 1g humid acid
    • 1.7g KCl (potassium chloride)
    • 0.5g Na2HPO4 (SUBSTITUE: K2HPO4)
    • 0.5g MgSO4 (magnesium sulfate)
    • 0.02g CaCO3 (calcium carbonate)
    • 0.01g FeSO4 (ferric sulfate)
    • 15g agar (can be 15g)
  5. Autoclave for 30 min at 121^o^C and 15 psi
  6. Pour plates
